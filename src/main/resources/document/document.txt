# 聚友GOGO平台知识库

## 一、平台简介

聚友GOGO是一个基于兴趣匹配的好友组队平台，旨在帮助用户找到志同道合的伙伴，共同参与活动、组建团队。平台核心功能包括：聊盆友（社区聊天）、找对伍（队伍管理）、同行去（拼车出行）、AI助手（智能问答）。

---

## 二、拼车功能

### 2.1 拼车功能概述

拼车功能允许用户基于地理位置创建或加入拼车队伍，实现出行成本分摊和环保出行。

### 2.2 发布拼车流程

1. **进入拼车页面**：点击"同行去"功能，选择"发布拼车"
2. **填写队伍信息**：
   - 队伍名称：为拼车队伍起一个清晰的名称
   - 队伍标签：**必须选择拼车相关标签**（拼车、景区往返、长途出行、节假日出行、上下班通勤），也可以添加自定义标签
   - 队伍描述：建议格式为"（出发地）->（目的地）时间点范围"，例如"（北京西站）->（首都机场）8:00-9:00"
   - 最大人数：至少2人，根据车辆座位数合理设置
   - 过期时间：选择拼车队伍的有效截止时间
3. **定位确认**：系统会自动获取当前位置，发布拼车后请避免大范围活动，避免定位误差
4. **提交创建**：确认信息无误后点击"创建队伍"

### 2.3 匹配拼车队伍

1. **开始拼车**：点击"开始拼车"按钮
2. **查看附近队伍**：系统会显示1公里范围内的拼车队伍
3. **刷新列表**：如果未看到合适队伍，可以点击"刷新"按钮重新加载
4. **加入队伍**：选择合适的队伍后，点击加入

### 2.4 拼车安全提醒

- **身份验证**：建议在加入拼车队伍前，通过平台聊天功能与车主沟通，了解对方基本信息
- **行程确认**：出发前再次确认出发时间、地点和路线
- **费用约定**：提前与车主沟通费用分摊方式，避免产生纠纷
- **位置共享**：建议与家人朋友分享行程信息
- **避免大范围移动**：发布拼车后保持相对固定位置，确保匹配准确性

### 2.5 费用分摊规则

平台不强制规定费用分摊方式，建议用户自行协商：

- **常见方式**：
  - 按人数平均分摊：总费用除以总人数
  - 按距离分摊：根据每人乘坐距离比例分摊
  - 车主免费：乘客分摊油费和过路费
- **建议**：出发前明确费用分摊方式，避免途中产生分歧
- **支付方式**：建议使用平台聊天功能确认后，通过线下或第三方支付平台完成

## 三、队伍管理

### 3.1 创建队伍

1. **进入创建页面**：点击"找对伍"功能，选择"创建队伍"
2. **填写队伍信息**：
   - 队伍名称：为队伍起一个吸引人的名称
   - 队伍标签：选择相关标签，支持自定义标签添加
   - 队伍描述：详细说明队伍的活动内容、规则等
   - 最大人数：设置队伍容纳的最大成员数
   - 队伍类型：
     - 公开：所有人可见可加入
     - 私有：仅邀请成员可见
     - 加密：需要密码才能加入
   - 过期时间：设置队伍的有效期
3. **提交创建**：创建成功后，创建者自动成为队长

### 3.2 加入队伍

- **公开队伍**：直接搜索并加入
- **私有队伍**：需要收到邀请链接
- **加密队伍**：需要输入正确的密码

### 3.3 队伍权限

- **队长**：创建者自动成为队长，拥有队伍管理权限
- **管理员**：平台管理员拥有所有权限
- **普通成员**：可以参与队伍活动和聊天

## 四、用户匹配

### 4.1 标签系统

- **标签作用**：标签是用户兴趣和特长的标识，用于匹配志同道合的伙伴
- **标签类型**：平台提供多种预设标签，用户也可以自定义标签
- **标签使用**：在个人资料中添加标签，系统会根据标签推荐相似用户和队伍

### 4.2 搜索用户

1. **进入搜索页面**：点击"搜索"功能
2. **输入关键词**：可以输入用户名、标签等关键词
3. **查看结果**：浏览匹配的用户列表
4. **发起聊天**：点击用户卡片可以查看详细信息并发起聊天

### 4.3 搜索队伍

1. **进入队伍页面**：点击"找对伍"功能
2. **搜索队伍**：可以按队伍名称、标签、描述等关键词搜索
3. **筛选条件**：可以根据标签、队伍类型等条件筛选
4. **加入队伍**：找到合适的队伍后点击加入

## 五、常见问题（FAQ）

### Q1: 为什么发布拼车时必须选择拼车标签？

A: 拼车标签是系统识别拼车队伍的关键标识，只有选择了拼车相关标签，队伍才会在拼车匹配功能中显示，其他用户才能通过"开始拼车"功能找到您的队伍。

### Q2: 拼车队伍匹配范围是多少？

A: 系统默认匹配1公里范围内的拼车队伍。如果您的定位不准确或移动范围过大，可能影响匹配结果，建议发布拼车后保持相对固定位置。

### Q3: 如何修改已创建的队伍信息？

A: 进入队伍详情页面，点击"编辑队伍"按钮，可以修改队伍名称、描述、最大人数、过期时间等信息。注意：队伍类型和密码创建后无法修改。

### Q4: 队伍过期后还能使用吗？

A: 队伍过期后，系统会自动隐藏该队伍，其他用户无法搜索到。但已加入的成员仍可以在队伍聊天中交流，只是无法再接受新成员。

### Q5: 如何退出队伍？

A: 进入队伍详情页面，点击"退出队伍"按钮即可。注意：队长退出队伍前，建议先转让队长权限或解散队伍。

### Q6: 标签可以自定义吗？

A: 可以。在创建队伍或编辑个人资料时，除了选择预设标签，还可以输入自定义标签并添加。

### Q7: 拼车费用如何支付？

A: 平台不提供支付功能，费用由用户自行协商和支付。建议在出发前通过平台聊天功能确认费用分摊方式，避免产生纠纷。

### Q8: 如何确保拼车安全？

A: 建议在加入拼车前，通过平台聊天功能与车主沟通，了解对方基本信息；出发前确认行程细节；与家人朋友分享行程信息；选择信誉良好的车主。

## 六、使用建议

1. **完善个人资料**：添加真实头像、详细描述和兴趣标签，有助于找到更匹配的伙伴
2. **合理设置标签**：选择准确反映自己兴趣的标签，避免过多或过少
3. **及时沟通**：加入队伍或拼车后，主动与其他成员沟通，建立信任
4. **遵守规则**：尊重队伍规则，按时参加活动，维护良好的社区氛围
5. **注意安全**：线下活动时注意人身和财产安全，特别是拼车出行
